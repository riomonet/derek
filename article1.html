Disguiser Json Parser

The goal is to be able to query files provide by the Disguiser Platform on research into internet censorhsip to print a list of request data beteween a certain data and time for any particuliar ANS. The requirements set by Professor Chen is to call the program with the following arguments.

Command Line Arguements

  • Name of Raw File from Disguiser Platform
  • Start Date/Time specify format.
  • End Date/Time specify format

Current Solution

I have only focused on http reqeusts. There are other files, SNI and DNS, records as well. I will have to adjust the process slightly for each file. The idea is to parse the json into c data structures and then serializ the data into a binary files.

Pre Processing

The parser loads the entire 3.2gb file into memory. The goal is to remove all json specicific symbols and end up with a single string of keys followed by values, all separated by null charavcter '/0'.
